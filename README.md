# AlexssanderLX Portfolio

Personal portfolio built with ASP.NET, showcasing development, offensive security labs, and original music compositions.

This project serves both as a professional presentation platform and as a continuous deployment and architecture training environment.

---

## About the Project

This portfolio was developed using:

- C#
- ASP.NET
- Entity Framework
- Responsive layout
- Structured project organization

It is designed to evolve over time, incorporating new backend features, security improvements, and deployment refinements.

---

## Sections

### Development
Projects focused on backend engineering, structured APIs, clean architecture, and fullstack implementations.

### Pentest
Structured lab exploitations and security challenges, demonstrating enumeration, vulnerability analysis, remote code execution, and privilege escalation techniques.

### Music
Original piano compositions and creative works, reflecting discipline, structure, and independent production.
